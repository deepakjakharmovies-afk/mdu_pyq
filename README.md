# MDU Previous Years Questions Project

## Overview
This project is designed to collect, organize, and make available previous years' questions for the MDU (Maharshi Dayanand University) examinations to help students prepare effectively.

## Features
- **Comprehensive Question Bank**: Contains questions from various subjects and years.
- **User-Friendly Search Functionality**: Allows users to easily find questions by subject or year.
- **Regular Updates**: Ensures that the question bank is frequently updated with the latest available questions.

## Installation
To install this project, follow the steps below:

1. Clone the repository:
   ```bash
   git clone https://github.com/deepakjakharmovies-afk/mdu_pyq.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mdu_pyq
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the project, use the following command:
```bash
python main.py
```
